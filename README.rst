===============================
Trellopy
===============================

A Python mockup of Trello with CLI. Trello is a very popular (and powerful) todo-list / task board. 

* Free software: BSD license
* Documentation: https://trellopy.readthedocs.org.

Requirements
============
* PyMongo (and of course, MongoDB)
* Nose (for testing)
